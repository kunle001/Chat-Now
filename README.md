# Chat Now

**Design Type:** Monolithic
**System Architecture:** Pending

## App Description
Chat Now is a node.js application that enables live chat between individuals and groups. All conversations are saved in a database.

## Functionalities Embedded
### 1. Authentication
- Sign up
- Login
- Logout
- Reset Password
- Confirmation Email

#### User Profile
- ID
- Avatar
- Display Full Name
- Email Address
- Username
- Category of Writer
- Bio

### 2. Rooms to Join
- ID
- Room Owner
- Date of Creation
- Room Title
- Pack Description
- CRUD Operations
- Room Participants
